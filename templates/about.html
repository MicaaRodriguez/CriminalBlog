About CriminalBlog

CriminalBlog es una aplicación web desarrollada con Django que funciona como un blog dedicado a casos criminales.

Los usuarios pueden explorar distintos casos, leer información detallada y participar en discusiones mediante comentarios.

Este proyecto fue desarrollado como proyecto final del curso de Django.

Tecnologías utilizadas:

  • Python
  • Django
  • SQLite
  • Bootstrap
