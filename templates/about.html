About CriminalBlog

CriminalBlog es una app web desarrollada con la finalidad de crear un blog para los amantes de los casos criminales.

En la mismo los usuarios pueden conocer distintos casos criminales, leer información detallada y participar en discusiones mediante comentarios.

Este proyecto fue desarrollado como proyecto final del curso de Python.

En el mismo se utilizaron las siguientes tecnologias:

  • Python
  • Django
  • SQLite
  • Bootstrap
